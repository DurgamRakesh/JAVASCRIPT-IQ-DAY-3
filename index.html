16.What are arrow functions?

        IN JS, Arrow functions are also called as "fat arrow"(=>) functions.
        -Arrow functions are new way of writing functions in js without using function keyword and return keyword.
        -It was introduced in ES6.
        -We are omit curly braces and function and return keyword.
        -Arrow functions are assigned to a variable.

        EX:
        let arrow = (a,b) => a*b;
        console.log(arrow(5,6));

18.What is callback hell?

        IN JS, Callback hell is function with nested callbacks which makes code hard to read and debug
        when dealing with ansychronus logic.
        -The callback hell looks like pyramid structure.
        EX:
        setTimeout(()=>{
            console.log(1);
            setTimeout(()=>{
                console.log(2);
                setTimeout(()=>{
                    console.log(3);
                },3000);
            },2000);
        },1000);

19.Please explain Self Invoking Function and its code?

        IN JS, An IIFE(immediatly Invoked function expression) is a javascript function that runs as soon
        as it declated.()().
        -Also known as self Invoking function anonymous function.

        EX:
        (function self(){
            console.log('this is IIFE function')
        })()

        -The primary reason for to use this IIFE is to obtain data privacy.
        because any variable declated with IIFE cannot accessed outside of function.
        if we try it will give error

20.What is the use of setTimeout?

        The setTimeout() method executes a function or block of code after the specified time expires.
        -The method execute the code only once.

        SYNTAX: setTimeout(function,time);

        EX:
        setTimeout(()=>{
            console.log('this is setTimeout function')
        },5000)
